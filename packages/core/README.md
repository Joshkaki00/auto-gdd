# @auto-gdd/core

Shared engine for [auto-gdd](https://github.com/auto-gdd/auto-gdd).

Exports: `WorkspaceDetector`, `ConfigStore`, `OllamaClient`, `EmbeddingClient`, `RAGIndexer`, `HybridRetriever`, `GDDAssembler`, `MarkdownWriter`, `WorkspaceScanner`, `CursorScaffold`, `ENGINE_PROFILES`.

This package is a peer dependency of `auto-gdd` (CLI) and `auto-gdd-mcp`. It is not intended to be used standalone, but the API is stable and public.
